# Public Boundaries

This kit can expose the shape of a Build Contract. It must not expose the private judgment engine that creates high-quality contracts from messy intent.

## Public-Safe

- Build Contract schema.
- Builder report protocol.
- Review result shape.
- Markdown formatter.
- Toy examples.
- Synthetic fixtures.
- Safety checks that reject obvious secret or private-field leaks.

## Private

- real Brieftify app source
- live LLM pipeline
- production prompts
- real classifier logic
- question scoring and repair
- semantic-contract extraction
- artifact repair templates
- hidden judgment scorecard thresholds
- private regression bank
- production auth/history/deployment setup

## Rule

If a file would help someone clone Brieftify's result quality instead of understanding the public contract primitive, keep it private.
